 🌊** TNPSC QUIZZY: The Liquid Learning Experience**

**TNPSC QUIZZY** is a high-octane, visually immersive examination prep platform designed to turn the rigors of TNPSC study into an engaging, "game-show" style experience. Moving away from static, dull interfaces, this web app features a cutting-edge **Liquid Crystal** design system—where vibrant sky blue and sun-drenched yellow fluid dynamics flow seamlessly behind a modern glassmorphic UI. 

Inspired by the energetic UX of platforms like Kahoot, **TNPSC QUIZZY** prioritizes smooth transitions and haptic-style interactions to keep aspirants focused and motivated.

 Key Features
* **Fluid Motion UI:** A custom-coded "Metaball" background that creates a living, breathing atmosphere while you study.
* **Community-Driven Content:** An intuitive, glass-layered **Quiz Upload** system that allows users to contribute their own TNPSC questions with a single click.
* **Staggered Animations:** Kahoot-style reveal animations that build excitement and improve information retention through visual cues.
* **Optimized for TNPSC:** Clean, high-contrast typography specifically curated to make complex Tamil Nadu Public Service Commission questions easy to read on any device.

--- Tech Stack
* **Frontend:** HTML5, CSS3 (Advanced Keyframe Animations & Filter Blurs)
* **Logic:** Vanilla JavaScript (ES6+)
* **Design:** Glassmorphism & Fluid Dynamics Engine
